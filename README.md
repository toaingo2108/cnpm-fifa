## Installation

### Clone git

```sh
git clone https://github.com/toaingo2108/CNPM-HCMUS.git
```

### Install client node_modules

```sh
cd client
npm install
```

### Install server node_modules

```sh
cd server
npm install
```

## Run app

```sh
cd server
npm run server
```

```sh
cd client
npm start
```
